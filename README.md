프로젝트명: MatjipON

프로젝트 개요
식당 대기 등록 및 관리 기능을 제공하는 웹 기반 서비스입니다.
사용자는 대기 등록과 키오스크 이용을 할 수 있고, 관리자는 대기 현황과 주문을 실시간으로 관리할 수 있습니다.

주요 기능
- 사용자 대기 등록 및 취소
- 대기 순번 자동 관리
- 대기 현황 조회
- 관리자용 대기 관리 기능
- 관리자용 리뷰 관리 기능
- 사용자용 키오스크 주문 기능
- 사용자용 리뷰, 별점 쓰기 기능

담당 역할
- Oracle Database 기반 대기 테이블 설계
- 주문 내역과 상세 내역 테이블 분리 구조 설계
- WAITING_ID 자동 정렬 로직 구현
- 실시간 별점 반영 기능 구현
- Java Swing과 Oracle DB 연동을 통한 데이터 출력 및 관리

기술 스택
Java / Java Swing / JDBC / Oracle Database / MVC Architecture / Eclipse IDE

개발 목적 및 배운 점
대기 순번 관리 로직을 구현하며,
데이터 정렬 기준과 시간 정보 관리가 시스템 안정성에 중요한 요소임을 학습했습니다.
